# CSE107-HSTU for ECE 26

Here you'll find all the C programs taught in the lab class of CSE107 course. You can view the code here or can download it in your PC.

# Important Links

## Code::Blocks
Link: https://www.codeblocks.org/downloads/binaries/

Or you can download with Powershell if you want to flex
```Powershell
winget install -e --id CodeBlocks.CodeBlocks.MinGW
```
Or
```Powershell
winget install -e --id CodeBlocks.CodeBlocks
```

## VS Code (Visual Studio Code)
Link: https://code.visualstudio.com/download

Powershell,
```Powershell
winget install -e --id Microsoft.VisualStudioCode
```
**Note: If you are using VS Code, you need to install the compiler separately, which for C is gcc. To install gcc compiler, you need to download MinGW.**

## MinGW
Link: https://sourceforge.net/projects/mingw/files/Installer/mingw-get-setup.exe/download

*Note: Download will start as soon as you enter the link*

Powershell,
```Powershell
winget install BrechtSanders.WinLibs.POSIX.UCRT
```
